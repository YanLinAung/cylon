Pebble

Repository Issues

The Pebble is a smart watch, that connects via bluetooth.

For more info about the Pebble platform click here.

How to install

Installing Cylon.js with pebble support is pretty easy. 

npm install cylon-pebble

How to use

This small program receives message queue events from a Pebble smart watch.

JavaScript

Cylon = require('cylon');

Cylon.api({
  host: '0.0.0.0',
  port: '8080'
});

robot = {
  connection: {
    name: 'pebble',
    adaptor: 'pebble'
  },
  device: {
    name: 'pebble',
    driver: 'pebble'
  },
  work: function(self) {
    var c;
    c = 0;
    return every(1..second(), function() {
      c++;
      self.pebble.message_queue().push(c);
      return console.log(self.pebble.last_message());
    });
  }
};

bot = Object.create(robot);

bot.name = 'pebble';

Cylon.robot(Object.create(bot));

Cylon.start();

How to connect

This software requires version 1.12.0 of the Pebble watch firmware.

OSX

The main steps are:

  • Pair your computer and the Pebble
  • Use a socket to serial connection to map a TCP socket to the local port
  • Connect to the device via Cylon.js

Ubuntu

The main steps are:

  • Pair your computer and the Pebble
  • Map your device to a port
  • Use a socket to serial connection to map a TCP socket to the local port
  • Connect to the device via Cylon.js
  • You will need to pair with the Pebble, entering any needed password.
