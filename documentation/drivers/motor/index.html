Motor

Driver

Provides motor manipulation tools for Cylon.js like on/off, speed modulation using PWM, use of bidirectional motors using forward/backward pins.

How to establish a connection and setup the driver

JavaScript

var Cylon = require('cylon');

Cylon.robot({
  connection: {name: 'arduino', adaptor: 'firmata', port: '/dev/ttyACM0'},

  device: {name: 'motor', driver: 'motor', pin: 3},
  ...

CoffeeScript

Cylon = require 'cylon'

Cylon.robot
  connection:
    name: 'arduino', adaptor: 'firmata', port: '/dev/ttyACM0'

  device:
    name: 'motor', driver: 'motor', pin: 3
  ...

Commands:

turnOn → true | nil
Starts the motor.
turnOff → true | nil
Stops the motor.
toggle → true | nil
Sets the state of the motor to the oposite of the current state, if motor is on then sets it to off.
speed(speed_value) → true | nil
Sets the speed of the motor to the value provided in the speed param, speed value must be an integer between 0 and 255.
currentSpeed → speed
Returns the current speed of the motor as an integer between 0 and 255.

Events

start
Sent when the device has been started and is ready to use.

How to use

This example turns on a DC motor using PWM.

JavaScript

var Cylon = require('cylon');

Cylon.robot({
  connection: {name: 'arduino', adaptor: 'firmata', port: '/dev/ttyACM0'},

  device: {name: 'motor', driver: 'motor', pin: 3},

  work: function(my) {
    my.motor.speed(100);

    after((5).seconds(), function(my) {
      my.motor.stop();
    });
  }
}).start();

CoffeeScript

Cylon = require 'cylon'

Cylon.robot

    name: 'arduino', adaptor: 'firmata', port: '/dev/ttyACM0'
