Crazyflie

Driver

Allows user to send flight control commands to a Crazyflie micro-quadcopter. Flight control commands are those used to tell the Crazyflie to take off, land, and or any number of other autonomous flight manuvers.

How to establish a connection and setup the driver

Communication with the Crazyflie's flight control interface takes place using a 2.4GHz radio connection. The Crazyflie is controlled from your computer using a Crazyradio, which is a custom USB 2.4GHz radio device.

JavaScript

var Cylon = require('cylon');

Cylon.robot({
  connection: { name: 'crazyflie', adaptor: 'crazyflie' },
  device: {name: 'drone', driver: 'crazyflie'},
...

Commands:

land
Initiates a landing sequence.
takeoff
Initiates take off sequence.
hover
Makes the crazyflie hover in a position.
setPitch
Orders crazyflie to go forward or backward by the amount specified by deg.
setYaw
Orders crazyflie to turn left or right by the amount specified by deg.
setThrust
Orders crazyflie to accelerate.
shutdown
Shutdown the device.
setParam(param, value)
Set the parameters for crazyflie.

Events:

start
Sent when the device has been started and is ready to use.
stop
Sent when the device has been stopped.
setParam(param, value)
Set the parameters for crazyflie.

How to use

Examples

This small program causes the Crazyflie to takeoff for 10 seconds.

JavaScript

var Cylon = require('cylon');

Cylon.robot({
  connection: { name: 'crazyflie', adaptor: 'crazyflie', port: 'radio://1/10/250KPS' },
  device: {name: 'drone', driver: 'crazyflie'},

  work: function(my) {
    my.drone.on('start', function() {
      my.drone.takeoff();
      after((10).seconds(), function() { 
        my.drone.land();
      });
      after((15).seconds(), function() { 
        my.drone.stop();
      });
    });
  }
}).start();
